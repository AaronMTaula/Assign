? Server Layer (PHP — webdev.aut.ac.nz/assign)
booking.php
  • Receives POST data from booking.js
  • Generates BRN (BRN00001 format)
  • Generates booking date/time + status="unassigned"
  • INSERT into MySQL bookings table
  • Returns JSON confirmation
admin.php
  • Receives search request (GET/POST)
  • If bsearch non-empty ? SELECT by BRN
  • If bsearch empty ? SELECT unassigned within 2hrs
  • Returns matching rows as JSON
  • On assign request ? UPDATE status="assigned"
  • Returns confirmation message

? Database Layer (MySQL — phpMyAdmin)
bookings table
  • booking_id INT AUTO_INCREMENT PK
  • brn VARCHAR(8) — e.g. BRN00001
  • cname VARCHAR — customer name
  • phone VARCHAR(12)
  • unumber VARCHAR — unit no. (opt)
  • snumber VARCHAR — street no.
  • stname VARCHAR — street name
  • sbname VARCHAR — pickup suburb
  • dsbname VARCHAR — dest. suburb
  • pickup_date DATE
  • pickup_time TIME
  • booking_datetime DATETIME (server-generated)
  • status ENUM('unassigned','assigned')
Data Flow — Step by Step
?? Booking Flow
1User fills booking.html form (name, phone, address, date, time)
2booking.js validates inputs client-side (NOT NULL, phone format, date not in past)
3JS sends async Fetch POST to booking.php
4PHP generates BRN, booking timestamp, status="unassigned"
5PHP INSERTs full record into MySQL bookings table
6PHP returns JSON with BRN, pickup date/time
7JS inserts confirmation into <p id="reference"> on same page
?? Admin Flow
1Admin opens admin.html, enters BRN or leaves bsearch empty
2admin.js validates BRN format if non-empty, else skips
3JS sends Fetch request to admin.php with search param
4PHP queries: exact BRN match OR unassigned pickups within 2hrs
5PHP returns JSON rows; JS builds HTML table in <div class="content">
6Admin clicks Assign button on a row
7JS sends BRN to admin.php ? PHP UPDATEs status ? returns confirmation
